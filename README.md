# Nuclear Escalation Simulator

> **Educational simulation — simplified heuristic models using public data. Not operational or predictive.**

An educational, browser-based 3D globe that visualizes the catastrophic *scale* of a nuclear exchange in order to illustrate why escalation has no winners (the "Mutually Assured Destruction" lesson).
